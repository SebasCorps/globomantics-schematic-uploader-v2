# Contributing to Globomantics Schematic Uploader

Thank you for considering contributing!

## How to Contribute

- Fork the repository and create your branch from `main`.
- Write clear, concise commit messages.
- Ensure your code passes linting and tests.
- Submit a pull request (PR) with a clear description.

## Code Style

- Use consistent formatting (see `.editorconfig` if present).
- Prefer descriptive variable and function names.
- Write comments where necessary, but keep code self-explanatory.

## Getting Help

If you have questions, open a GitHub Discussion or email timothy@timothywarner.org.

## Code of Conduct

By participating, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).
